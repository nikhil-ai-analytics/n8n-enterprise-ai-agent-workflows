## **‚ñ∂Ô∏è How to Run This Project**

### **Step 1: Install n8n**

Set up n8n in your preferred environment:

- Install locally using Node.js  
- Deploy via Docker  
- Use n8n Cloud (recommended for enterprise use)

Ensure n8n is properly configured and accessible before proceeding.

---

### **Step 2: Navigate to the Desired Workflow Folder**

Browse this repository and navigate to the specific folder corresponding to the enterprise AI agent workflow you want to deploy.

Each folder represents a dedicated automation agent designed for a specific enterprise use case.

---

### **Step 3: Download the Workflow JSON File**

Inside the selected folder:

- Locate the relevant `.json` workflow file  
- Download the file to your local system  

This file contains the complete automation logic and AI agent configuration.

---

### **Step 4: Import the Workflow into n8n**

- Open your n8n dashboard  
- Click on **Import Workflow**  
- Upload the downloaded `.json` file  
- Review the workflow structure after import  

Once imported, configure the required credentials and activate the workflow.

---

After activation, test the workflow with live inputs (webhooks, emails, voice triggers, etc.) and monitor execution logs for validation.
